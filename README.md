# Portfolio Website

Personal portfolio showcasing my skills, projects, and experience as a web designer and developer.

## Features

- Responsive single-page design
- Dark mode with 5 color themes
- Smooth scrolling navigation
- Animated typing effect
- Skills showcase with timeline

## Tech Stack

- HTML5, CSS3, JavaScript
- Font Awesome icons
- Typed.js animations
- Google Fonts (Poppins, Clicker Script)

## Quick Start

```bash
git clone https://github.com/theurgicthaps/portfolio-website.git
cd portfolio-website
open index.html
```

## Sections

- **Home** - Introduction and overview
- **About** - Skills, education, and experience
- **Services** - Web design, development, and more
- **Portfolio** - Project showcase
- **Contact** - Get in touch
